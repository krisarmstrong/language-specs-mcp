# WCAG H36: Require alt text on images used as submit buttons

Rule ID:wcag/h36Category:AccessibilityStandards:

- WCAG 2.2 (A)
- WCAG 2.1 (A)
- WCAG 2.0 (A)

[WCAG H36](https://www.w3.org/WAI/WCAG22/Techniques/html/H36) requires all images used as submit buttons to have a non-empty textual description using the `alt` attribute. The text must indicate the buttons function but not describe the image.

See [wcag/h37](h37.html) for a similar rule for `<img>`.

## [Rule details](#rule-details)

Examples of incorrect code for this rule:

```
<!-- missing alt attribute -->
<input type="image" src="submit-button.png">

<!-- empty alt attribute -->
<input type="image" src="submit-button.png" alt="">
```
