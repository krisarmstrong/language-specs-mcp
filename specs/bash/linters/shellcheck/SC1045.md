# It's not `foo &; bar`, just `foo & bar`.

### Problematic code:

```
#cb1-1foo &; bar
```

### Correct code:

```
#cb2-1foo & bar
```

### Rationale:

Both `&` and `;` terminate the command. You should only use one of them.

### Exceptions

None.
