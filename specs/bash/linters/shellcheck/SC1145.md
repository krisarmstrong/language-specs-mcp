## Unknown `external-sources` value. Expected `true`/`false`.

### Problematic code:

`.shellcheckrc`:

```
#cb1-1external-sources=maybe
```

### Correct code:

```
#cb2-1external-sources=true
```

### Rationale:

The `external-sources`[directive](directive) expects a value of `true` or `false`.

### Exceptions:

None.
