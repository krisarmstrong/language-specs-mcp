## Did you forget the `;;` after the previous case item?

### Problematic code:

```
#cb1-1while getoptions f option
#cb1-2do
#cb1-3  case "${options}"
#cb1-4  in
#cb1-5  f) FTR="${ARG}"
#cb1-6    \?) exit
#cb1-7  esac
#cb1-8done
```

### Correct code:

```
#cb2-1while getoptions f option
#cb2-2do
#cb2-3  case "${options}"
#cb2-4  in
#cb2-5  f) FTR="${ARG}";;
#cb2-6    \?) exit;;
#cb2-7  esac
#cb2-8done
```

### Rationale:

Syntax `case` needs `;;` after the previous case item. If not, syntax error will cause.
