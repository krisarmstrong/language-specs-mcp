## Remove spaces between (( in arithmetic for loop.

### Problematic code:

```
#cb1-1for( (i=0; i<10; i++) ); do echo $i; done
```

### Correct code:

```
#cb2-1for((i=0; i<10; i++)); do echo $i; done
```

### Rationale:

ShellCheck finds arithmetic for ((;;)) expressions where (( or )) are intervening with spaces

### Exceptions:

None
