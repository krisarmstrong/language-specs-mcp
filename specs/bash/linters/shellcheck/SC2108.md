## In `[..](..)`, use `&&` instead of `-a`.

### Problematic code:

```
#cb1-1[[ "$1" = "-v" -a -z "$2" ]]
```

### Correct code:

```
#cb2-1[[ "$1" = "-v" && -z "$2" ]]
```

### Rationale:

`-a` for logical AND is not supported in a `[[ .. ]]` expression. Use `&&` instead.

### Exceptions:

None.
