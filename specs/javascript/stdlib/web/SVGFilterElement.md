# SVGFilterElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGFilterElement&level=high)

The `SVGFilterElement` interface provides access to the properties of [<filter>](/en-US/docs/Web/SVG/Reference/Element/filter) elements, as well as methods to manipulate them.

## [Instance properties](#instance_properties)

[SVGFilterElement.href](/en-US/docs/Web/API/SVGFilterElement/href)Read only

An [SVGAnimatedString](/en-US/docs/Web/API/SVGAnimatedString) that corresponds to the [href](/en-US/docs/Web/SVG/Reference/Attribute/href) or [xlink:href](/en-US/docs/Web/SVG/Reference/Attribute/xlink:href)Deprecated attribute of the given [<filter>](/en-US/docs/Web/SVG/Reference/Element/filter) element.

[SVGFilterElement.filterUnits](/en-US/docs/Web/API/SVGFilterElement/filterUnits)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) that corresponds to the [filterUnits](/en-US/docs/Web/SVG/Reference/Attribute/filterUnits) attribute of the given [<filter>](/en-US/docs/Web/SVG/Reference/Element/filter) element. Takes one of the constants defined in [SVGUnitTypes](/en-US/docs/Web/API/SVGUnitTypes).

[SVGFilterElement.primitiveUnits](/en-US/docs/Web/API/SVGFilterElement/primitiveUnits)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) that corresponds to the [primitiveUnits](/en-US/docs/Web/SVG/Reference/Attribute/primitiveUnits) attribute of the given [<filter>](/en-US/docs/Web/SVG/Reference/Element/filter) element. Takes one of the constants defined in [SVGUnitTypes](/en-US/docs/Web/API/SVGUnitTypes).

[SVGFilterElement.x](/en-US/docs/Web/API/SVGFilterElement/x)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) that corresponds to the [x](/en-US/docs/Web/SVG/Reference/Attribute/x) attribute on the given [<filter>](/en-US/docs/Web/SVG/Reference/Element/filter) element.

[SVGFilterElement.y](/en-US/docs/Web/API/SVGFilterElement/y)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) that corresponds to the [y](/en-US/docs/Web/SVG/Reference/Attribute/y) attribute of the given [<filter>](/en-US/docs/Web/SVG/Reference/Element/filter) element.

[SVGFilterElement.width](/en-US/docs/Web/API/SVGFilterElement/width)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) that corresponds to the [width](/en-US/docs/Web/SVG/Reference/Attribute/width) attribute of the given [<filter>](/en-US/docs/Web/SVG/Reference/Element/filter) element.

[SVGFilterElement.height](/en-US/docs/Web/API/SVGFilterElement/height)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) that corresponds to the [height](/en-US/docs/Web/SVG/Reference/Attribute/height) attribute of the given [<filter>](/en-US/docs/Web/SVG/Reference/Element/filter) element.
