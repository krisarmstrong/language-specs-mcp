# SVGFEDisplacementMapElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGFEDisplacementMapElement&level=high)

The `SVGFEDisplacementMapElement` interface corresponds to the [<feDisplacementMap>](/en-US/docs/Web/SVG/Reference/Element/feDisplacementMap) element.

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent interface, [SVGElement](/en-US/docs/Web/API/SVGElement).

[SVGFEDisplacementMapElement.height](/en-US/docs/Web/API/SVGFEDisplacementMapElement/height)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [height](/en-US/docs/Web/SVG/Reference/Attribute/height) attribute of the given element.

[SVGFEDisplacementMapElement.in1](/en-US/docs/Web/API/SVGFEDisplacementMapElement/in1)Read only

[SVGFEDisplacementMapElement.in2](/en-US/docs/Web/API/SVGFEDisplacementMapElement/in2)Read only

[SVGFEDisplacementMapElement.result](/en-US/docs/Web/API/SVGFEDisplacementMapElement/result)Read only

An [SVGAnimatedString](/en-US/docs/Web/API/SVGAnimatedString) corresponding to the [result](/en-US/docs/Web/SVG/Reference/Attribute/result) attribute of the given element.

[SVGFEDisplacementMapElement.scale](/en-US/docs/Web/API/SVGFEDisplacementMapElement/scale)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [scale](/en-US/docs/Web/SVG/Reference/Attribute/scale) attribute of the given element.

[SVGFEDisplacementMapElement.width](/en-US/docs/Web/API/SVGFEDisplacementMapElement/width)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [width](/en-US/docs/Web/SVG/Reference/Attribute/width) attribute of the given element.

[SVGFEDisplacementMapElement.x](/en-US/docs/Web/API/SVGFEDisplacementMapElement/x)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [x](/en-US/docs/Web/SVG/Reference/Attribute/x) attribute of the given element.

[SVGFEDisplacementMapElement.xChannelSelector](/en-US/docs/Web/API/SVGFEDisplacementMapElement/xChannelSelector)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [xChannelSelector](/en-US/docs/Web/SVG/Reference/Attribute/xChannelSelector) attribute of the given element. It takes one of the `SVG_CHANNEL_*` constants defined on this interface.

[SVGFEDisplacementMapElement.y](/en-US/docs/Web/API/SVGFEDisplacementMapElement/y)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [y](/en-US/docs/Web/SVG/Reference/Attribute/y) attribute of the given element.

[SVGFEDisplacementMapElement.yChannelSelector](/en-US/docs/Web/API/SVGFEDisplacementMapElement/yChannelSelector)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [yChannelSelector](/en-US/docs/Web/SVG/Reference/Attribute/yChannelSelector) attribute of the given element. It takes one of the `SVG_CHANNEL_*` constants defined on this interface.

## [Instance methods](#instance_methods)

This interface does not provide any specific methods, but implements those of its parent, [SVGElement](/en-US/docs/Web/API/SVGElement).

## [Static properties](#static_properties)

[SVG_CHANNEL_UNKNOWN (0)](#svg_channel_unknown)

The type is not one of predefined types. It is invalid to attempt to define a new value of this type or to attempt to switch an existing value to this type.

[SVG_CHANNEL_R (1)](#svg_channel_r)

Corresponds to the value `R`.

[SVG_CHANNEL_G (2)](#svg_channel_g)

Corresponds to the value `G`.

[SVG_CHANNEL_B (3)](#svg_channel_b)

Corresponds to the value `B`.

[SVG_CHANNEL_A (4)](#svg_channel_a)

Corresponds to the value `A`.
