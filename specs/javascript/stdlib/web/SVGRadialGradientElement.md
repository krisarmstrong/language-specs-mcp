# SVGRadialGradientElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGRadialGradientElement&level=high)

The `SVGRadialGradientElement` interface corresponds to the [<RadialGradient>](/en-US/docs/Web/SVG/Reference/Element/radialGradient) element.

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent, [SVGGradientElement](/en-US/docs/Web/API/SVGGradientElement).

[SVGRadialGradientElement.cx](/en-US/docs/Web/API/SVGRadialGradientElement/cx)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [cx](/en-US/docs/Web/SVG/Reference/Attribute/cx) attribute of the given [<RadialGradient>](/en-US/docs/Web/SVG/Reference/Element/radialGradient) element.

[SVGRadialGradientElement.cy](/en-US/docs/Web/API/SVGRadialGradientElement/cy)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [cy](/en-US/docs/Web/SVG/Reference/Attribute/cy) attribute of the given [<RadialGradient>](/en-US/docs/Web/SVG/Reference/Element/radialGradient) element.

[SVGRadialGradientElement.r](/en-US/docs/Web/API/SVGRadialGradientElement/r)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [r](/en-US/docs/Web/SVG/Reference/Attribute/r) attribute of the given [<RadialGradient>](/en-US/docs/Web/SVG/Reference/Element/radialGradient) element.

[SVGRadialGradientElement.fr](/en-US/docs/Web/API/SVGRadialGradientElement/fr)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [fr](/en-US/docs/Web/SVG/Reference/Attribute/fr) attribute.

[SVGRadialGradientElement.fx](/en-US/docs/Web/API/SVGRadialGradientElement/fx)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [fx](/en-US/docs/Web/SVG/Reference/Attribute/fx) attribute of the given [<RadialGradient>](/en-US/docs/Web/SVG/Reference/Element/radialGradient) element.

[SVGRadialGradientElement.fy](/en-US/docs/Web/API/SVGRadialGradientElement/fy)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [fy](/en-US/docs/Web/SVG/Reference/Attribute/fy) attribute of the given [<RadialGradient>](/en-US/docs/Web/SVG/Reference/Element/radialGradient) element.

## [Instance methods](#instance_methods)

This interface doesn't implement any specific methods, but inherits methods from its parent interface, [SVGGradientElement](/en-US/docs/Web/API/SVGGradientElement).
