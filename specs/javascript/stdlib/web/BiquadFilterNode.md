An all-pass filter is a [signal processing filter](/wiki/Filter_(signal_processing)) that passes all [frequencies](/wiki/Frequency) equally in gain, but changes the [phase](/wiki/Phase_(waves)) relationship among various frequencies. Most types of filter reduce the amplitude (i.e. the magnitude) of the signal applied to it for some values of frequency, whereas the all-pass filter allows all frequencies through without changes in level. 

## Common applications

[[edit](/w/index.php?title=All-pass_filter&action=edit&section=1)]

A common application in [electronic music](/wiki/Electronic_music) production is in the design of an effects unit known as a "[phaser](/wiki/Phaser_(effect))", where a number of all-pass filters are connected in sequence and the output mixed with the raw signal. 

It does this by varying its [phase](/wiki/Phase_(waves)) shift as a function of frequency. Generally, the filter is described by the frequency at which the [phase shift](/wiki/Phase_shifting) crosses 90° (i.e., when the input and output signals go into [quadrature](/wiki/Quadrature_phase) – when there is a quarter [wavelength](/wiki/Wavelength) of delay between them). 

They are generally used to compensate for other undesired phase shifts that arise in the system, or for mixing with an unshifted version of the original to implement a notch [comb filter](/wiki/Comb_filter). 

They may also be used to convert a [mixed phase](/wiki/Minimum_phase#Mixed_phase) filter into a [minimum phase](/wiki/Minimum_phase) filter with an equivalent magnitude response or an unstable filter into a stable filter with an equivalent magnitude response.

### Implementation using low-pass filter

[[edit](/w/index.php?title=All-pass_filter&action=edit&section=3)]/wiki/File:Schem_All-Pass_Filter_Producing_Lag.pngAn op-amp base all-pass filter incorporating a low-pass filter.

The [operational amplifier](/wiki/Operational_amplifier) circuit shown in adjacent figure implements a single-pole [active](/wiki/Passivity_(engineering)) all-pass filter that features a [low-pass filter](/wiki/Low-pass_filter) at the non-inverting input of the opamp. The filter's [transfer function](/wiki/Transfer_function) is given by: 

H(s)=−s−1RCs+1RC=1−sRC1+sRC,{\displaystyle H(s)=-{\frac {s-{\frac {1}{RC}}}{s+{\frac {1}{RC}}}}={\frac {1-sRC}{1+sRC}},\,}

which has one [pole](/wiki/Pole_(complex_analysis)) at -1/RC and one [zero](/wiki/Zero_(complex_analysis)) at 1/RC (i.e., they are reflections of each other across the [imaginary](/wiki/Imaginary_number) axis of the [complex plane](/wiki/Complex_plane)). The [magnitude and phase](/wiki/Complex_plane) of H(iω) for some [angular frequency](/wiki/Angular_frequency) ω are 

|H(iω)|=1and∠H(iω)=−2arctan⁡(ωRC).{\displaystyle |H(i\omega )|=1\quad {\text{and}}\quad \angle H(i\omega )=-2\arctan(\omega RC).\,}

The filter has [unity](/wiki/Unity_(mathematics))-[gain](/wiki/Gain_(electronics)) magnitude for all ω. The filter introduces a different delay at each frequency and reaches input-to-output quadrature at ω=1/RC (i.e., phase shift is 90°).[[2]](#cite_note-2)

This implementation uses a low-pass filter at the [non-inverting input](/wiki/Operational_amplifier#Circuit_notation) to generate the phase shift and [negative feedback](/wiki/Negative_feedback). 

- At high [frequencies](/wiki/Frequency), the [capacitor](/wiki/Capacitor) is a [short circuit](/wiki/Short_circuit), creating an [inverting amplifier](/wiki/Operational_amplifier_applications#Inverting_amplifier) (i.e., 180° phase shift) with unity gain.
- At low frequencies and [DC](/wiki/DC_offset), the capacitor is an [open circuit](https://en.wiktionary.org/wiki/open_circuit), creating a [unity](/wiki/Unity_(mathematics))-[gain](/wiki/Gain_(electronics))[voltage buffer](/wiki/Operational_amplifier_applications#Voltage_follower) (i.e., no phase shift).
- At the [corner frequency](/wiki/Corner_frequency) ω=1/RC of the low-pass filter (i.e., when input frequency is 1/(2πRC)), the circuit introduces a 90° shift (i.e., output is in quadrature with input; the output appears to be delayed by a quarter [period](/wiki/Frequency) from the input).

In fact, the phase shift of the all-pass filter is double the phase shift of the low-pass filter at its non-inverting input. 

#### Interpretation as a Padé approximation to a pure delay

[[edit](/w/index.php?title=All-pass_filter&action=edit&section=4)]

The Laplace transform of a pure delay is given by 

e−sT,{\displaystyle e^{-sT},}

where T{\displaystyle T} is the delay (in seconds) and s∈C{\displaystyle s\in \mathbb {C} } is complex frequency. This can be approximated using a [Padé approximant](/wiki/Pad%C3%A9_approximant), as follows: 

e−sT=e−sT/2esT/2≈1−sT/21+sT/2,{\displaystyle e^{-sT}={\frac {e^{-sT/2}}{e^{sT/2}}}\approx {\frac {1-sT/2}{1+sT/2}},}

where the last step was achieved via a first-order [Taylor series](/wiki/Taylor_series) expansion of the numerator and denominator. By setting RC=T/2{\displaystyle RC=T/2} we recover H(s){\displaystyle H(s)} from above. 

### Implementation using high-pass filter

[[edit](/w/index.php?title=All-pass_filter&action=edit&section=5)]/wiki/File:Active_Allpass_Filter.svgAn op-amp base all-pass filter incorporating a high-pass filter.

The [operational amplifier](/wiki/Operational_amplifier) circuit shown in the adjacent figure implements a single-pole [active](/wiki/Passivity_(engineering)) all-pass filter that features a [high-pass filter](/wiki/High-pass_filter) at the non-inverting input of the opamp. The filter's [transfer function](/wiki/Transfer_function) is given by: 

H(s)=s−1RCs+1RC,{\displaystyle H(s)={\frac {s-{\frac {1}{RC}}}{s+{\frac {1}{RC}}}},\,}[[3]](#cite_note-3)

which has one [pole](/wiki/Pole_(complex_analysis)) at -1/RC and one [zero](/wiki/Zero_(complex_analysis)) at 1/RC (i.e., they are reflections of each other across the [imaginary](/wiki/Imaginary_number) axis of the [complex plane](/wiki/Complex_plane)). The [magnitude and phase](/wiki/Complex_plane) of H(iω) for some [angular frequency](/wiki/Angular_frequency) ω are 

|H(iω)|=1and∠H(iω)=π−2arctan⁡(ωRC).{\displaystyle |H(i\omega )|=1\quad {\text{and}}\quad \angle H(i\omega )=\pi -2\arctan(\omega RC).\,}

The filter has [unity](/wiki/Unity_(mathematics))-[gain](/wiki/Gain_(electronics)) magnitude for all ω. The filter introduces a different delay at each frequency and reaches input-to-output quadrature at ω=1/RC (i.e., phase lead is 90°). 

This implementation uses a [high-pass filter](/wiki/High-pass_filter) at the [non-inverting input](/wiki/Operational_amplifier#Circuit_notation) to generate the phase shift and [negative feedback](/wiki/Negative_feedback). 

- At high [frequencies](/wiki/Frequency), the [capacitor](/wiki/Capacitor) is a [short circuit](/wiki/Short_circuit), thereby creating a [unity](/wiki/Unity_(mathematics))-[gain](/wiki/Gain_(electronics))[voltage buffer](/wiki/Operational_amplifier_applications#Voltage_follower) (i.e., no phase lead).
- At low frequencies and [DC](/wiki/DC_offset), the capacitor is an [open circuit](https://en.wiktionary.org/wiki/open_circuit) and the circuit is an [inverting amplifier](/wiki/Operational_amplifier_applications#Inverting_amplifier) (i.e., 180° phase lead) with unity gain.
- At the [corner frequency](/wiki/Corner_frequency) ω=1/RC of the high-pass filter (i.e., when input frequency is 1/(2πRC)), the circuit introduces a 90° phase lead (i.e., output is in quadrature with input; the output appears to be advanced by a quarter [period](/wiki/Frequency) from the input).

In fact, the phase shift of the all-pass filter is double the phase shift of the high-pass filter at its non-inverting input. 

### Voltage controlled implementation

[[edit](/w/index.php?title=All-pass_filter&action=edit&section=6)]

The resistor can be replaced with a [FET](/wiki/Field-effect_transistor) in its ohmic mode to implement a voltage-controlled phase shifter; the voltage on the gate adjusts the phase shift. In electronic music, a [phaser](/wiki/Phaser_(effect)) typically consists of two, four or six of these phase-shifting sections connected in tandem and summed with the original. A low-frequency oscillator ([LFO](/wiki/Low-frequency_oscillation)) ramps the control voltage to produce the characteristic swooshing sound. 

## Passive analog implementation

[[edit](/w/index.php?title=All-pass_filter&action=edit&section=7)]

The benefit to implementing all-pass filters with [active components](/wiki/Passivity_(engineering)) like [operational amplifiers](/wiki/Operational_amplifiers) is that they do not require [inductors](/wiki/Inductor), which are bulky and costly in [integrated circuit](/wiki/Integrated_circuit) designs. In other applications where inductors are readily available, all-pass filters can be implemented entirely without active components. There are a number of circuit [topologies](/wiki/Topology_(electronics)) that can be used for this. The following are the most commonly used circuits. 

### Lattice filter

[[edit](/w/index.php?title=All-pass_filter&action=edit&section=8)]/wiki/File:Lattice_filter,_low_end_correction.svgAn all-pass filter using lattice topologyMain article: [Lattice phase equaliser](/wiki/Lattice_phase_equaliser)

The lattice phase equaliser, or filter, is a filter composed of lattice, or X-sections. With single element branches it can produce a phase shift up to 180°, and with resonant branches it can produce phase shifts up to 360°. The filter is an example of a [constant-resistance network](/wiki/Constant-resistance_network) (i.e., its [image impedance](/wiki/Image_impedance) is constant over all frequencies). 

### T-section filter

[[edit](/w/index.php?title=All-pass_filter&action=edit&section=9)]

The phase equaliser based on T topology is the unbalanced equivalent of the lattice filter and has the same phase response. While the circuit diagram may look like a low pass filter it is different in that the two inductor branches are mutually coupled. This results in transformer action between the two inductors and an all-pass response even at high frequency. 

### Bridged T-section filter

[[edit](/w/index.php?title=All-pass_filter&action=edit&section=10)]Main article: [Bridged T delay equaliser](/wiki/Bridged_T_delay_equaliser)

The bridged T topology is used for delay equalisation, particularly the differential delay between two [landlines](/wiki/Landline) being used for [stereophonic sound](/wiki/Stereophonic_sound) broadcasts. This application requires that the filter has a [linear phase](/wiki/Linear_phase) response with frequency (i.e., constant [group delay](/wiki/Group_delay)) over a wide bandwidth and is the reason for choosing this topology. 

## Digital implementation

[[edit](/w/index.php?title=All-pass_filter&action=edit&section=11)]

A [Z-transform](/wiki/Z-transform) implementation of an all-pass filter with a complex pole at z0{\displaystyle z_{0}} is 

H(z)=z−1−z0¯1−z0z−1{\displaystyle H(z)={\frac {z^{-1}-{\overline {z_{0}}}}{1-z_{0}z^{-1}}}\ }

which has a zero at 1/z0¯{\displaystyle 1/{\overline {z_{0}}}}, where z¯{\displaystyle {\overline {z}}} denotes the [complex conjugate](/wiki/Complex_conjugate). The pole and zero sit at the same angle but have reciprocal magnitudes (i.e., they are reflections of each other across the boundary of the [complex](/wiki/Complex_plane)[unit circle](/wiki/Unit_circle)). The placement of this pole-zero pair for a given z0{\displaystyle z_{0}} can be rotated in the complex plane by any angle and retain its all-pass magnitude characteristic. Complex pole-zero pairs in all-pass filters help control the frequency where phase shifts occur. 

To create an all-pass implementation with real coefficients, the complex all-pass filter can be cascaded with an all-pass that substitutes z0¯{\displaystyle {\overline {z_{0}}}} for z0{\displaystyle z_{0}}, leading to the [Z-transform](/wiki/Z-transform) implementation 

H(z)=z−1−z0¯1−z0z−1×z−1−z01−z0¯z−1=z−2−2ℜ(z0)z−1+|z0|21−2ℜ(z0)z−1+|z0|2z−2,{\displaystyle H(z)={\frac {z^{-1}-{\overline {z_{0}}}}{1-z_{0}z^{-1}}}\times {\frac {z^{-1}-z_{0}}{1-{\overline {z_{0}}}z^{-1}}}={\frac {z^{-2}-2\Re (z_{0})z^{-1}+\left|{z_{0}}\right|^{2}}{1-2\Re (z_{0})z^{-1}+\left|z_{0}\right|^{2}z^{-2}}},\ }

which is equivalent to the [difference equation](/wiki/Recurrence_relation)

y[k]−2ℜ(z0)y[k−1]+|z0|2y[k−2]=x[k−2]−2ℜ(z0)x[k−1]+|z0|2x[k],{\displaystyle y[k]-2\Re (z_{0})y[k-1]+\left|z_{0}\right|^{2}y[k-2]=x[k-2]-2\Re (z_{0})x[k-1]+\left|z_{0}\right|^{2}x[k],\,}

where y[k]{\displaystyle y[k]} is the output and x[k]{\displaystyle x[k]} is the input at discrete time step k{\displaystyle k}. 

Filters such as the above can be cascaded with [unstable](/wiki/Control_theory#Stability) or mixed-phase filters to create a stable or minimum-phase filter without changing the magnitude response of the system. For example, by proper choice of z0{\displaystyle z_{0}}, a pole of an unstable system that is outside of the [unit circle](/wiki/Unit_circle) can be canceled and reflected inside the unit circle.
