# SVGClipPathElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGClipPathElement&level=high)

The `SVGClipPathElement` interface provides access to the properties of [<clipPath>](/en-US/docs/Web/SVG/Reference/Element/clipPath) elements, as well as methods to manipulate them.

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent, [SVGElement](/en-US/docs/Web/API/SVGElement).

[SVGClipPathElement.clipPathUnits](/en-US/docs/Web/API/SVGClipPathElement/clipPathUnits)Read only

Returns an [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [clipPathUnits](/en-US/docs/Web/SVG/Reference/Attribute/clipPathUnits) attribute of the associated [<clipPath>](/en-US/docs/Web/SVG/Reference/Element/clipPath) element. Takes one of the constants defined in [SVGUnitTypes](/en-US/docs/Web/API/SVGUnitTypes).

[SVGClipPathElement.transform](/en-US/docs/Web/API/SVGClipPathElement/transform)Read only

Returns an [SVGAnimatedTransformList](/en-US/docs/Web/API/SVGAnimatedTransformList) corresponding to the [transform](/en-US/docs/Web/SVG/Reference/Attribute/transform) attribute of the associated [<clipPath>](/en-US/docs/Web/SVG/Reference/Element/clipPath) element.

## [Instance methods](#instance_methods)

This interface doesn't implement any specific methods, but inherits methods from its parent, [SVGElement](/en-US/docs/Web/API/SVGElement).
