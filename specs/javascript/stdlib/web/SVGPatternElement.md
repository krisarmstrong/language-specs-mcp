# SVGPatternElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGPatternElement&level=high)

The `SVGPatternElement` interface corresponds to the [<pattern>](/en-US/docs/Web/SVG/Reference/Element/pattern) element.

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent, [SVGElement](/en-US/docs/Web/API/SVGElement).

[SVGPatternElement.href](/en-US/docs/Web/API/SVGPatternElement/href)Read only

An [SVGAnimatedString](/en-US/docs/Web/API/SVGAnimatedString) corresponding to the [href](/en-US/docs/Web/SVG/Reference/Attribute/href) or [xlink:href](/en-US/docs/Web/SVG/Reference/Attribute/xlink:href)Deprecated attribute of the given [<pattern>](/en-US/docs/Web/SVG/Reference/Element/pattern) element.

[SVGPatternElement.patternUnits](/en-US/docs/Web/API/SVGPatternElement/patternUnits)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [patternUnits](/en-US/docs/Web/SVG/Reference/Attribute/patternUnits) attribute of the given [<pattern>](/en-US/docs/Web/SVG/Reference/Element/pattern) element. Takes one of the constants defined in [SVGUnitTypes](/en-US/docs/Web/API/SVGUnitTypes).

[SVGPatternElement.patternContentUnits](/en-US/docs/Web/API/SVGPatternElement/patternContentUnits)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [patternContentUnits](/en-US/docs/Web/SVG/Reference/Attribute/patternContentUnits) attribute of the given [<pattern>](/en-US/docs/Web/SVG/Reference/Element/pattern) element. Takes one of the constants defined in [SVGUnitTypes](/en-US/docs/Web/API/SVGUnitTypes).

[SVGPatternElement.patternTransform](/en-US/docs/Web/API/SVGPatternElement/patternTransform)Read only

An [SVGAnimatedTransformList](/en-US/docs/Web/API/SVGAnimatedTransformList) corresponding to the [patternTransform](/en-US/docs/Web/SVG/Reference/Attribute/patternTransform) attribute of the given [<pattern>](/en-US/docs/Web/SVG/Reference/Element/pattern) element.

[SVGPatternElement.viewBox](/en-US/docs/Web/API/SVGPatternElement/viewBox)Read only

An [SVGAnimatedRect](/en-US/docs/Web/API/SVGAnimatedRect) corresponding to the [viewBox](/en-US/docs/Web/SVG/Reference/Attribute/viewBox) attribute of the given [<pattern>](/en-US/docs/Web/SVG/Reference/Element/pattern) element.

[SVGPatternElement.preserveAspectRatio](/en-US/docs/Web/API/SVGPatternElement/preserveAspectRatio)Read only

An [SVGAnimatedPreserveAspectRatio](/en-US/docs/Web/API/SVGAnimatedPreserveAspectRatio) corresponding to the [preserveAspectRatio](/en-US/docs/Web/SVG/Reference/Attribute/preserveAspectRatio) attribute of the given [<pattern>](/en-US/docs/Web/SVG/Reference/Element/pattern) element.

[SVGPatternElement.x](/en-US/docs/Web/API/SVGPatternElement/x)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [x](/en-US/docs/Web/SVG/Reference/Attribute/x) attribute of the given [<pattern>](/en-US/docs/Web/SVG/Reference/Element/pattern) element.

[SVGPatternElement.y](/en-US/docs/Web/API/SVGPatternElement/y)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [y](/en-US/docs/Web/SVG/Reference/Attribute/y) attribute of the given [<pattern>](/en-US/docs/Web/SVG/Reference/Element/pattern) element.

[SVGPatternElement.width](/en-US/docs/Web/API/SVGPatternElement/width)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [width](/en-US/docs/Web/SVG/Reference/Attribute/width) attribute of the given [<pattern>](/en-US/docs/Web/SVG/Reference/Element/pattern) element.

[SVGPatternElement.height](/en-US/docs/Web/API/SVGPatternElement/height)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [height](/en-US/docs/Web/SVG/Reference/Attribute/height) attribute of the given [<pattern>](/en-US/docs/Web/SVG/Reference/Element/pattern) element.

## [Instance methods](#instance_methods)

This interface doesn't implement any specific methods, but inherits methods from its parent interface, [SVGElement](/en-US/docs/Web/API/SVGElement).
