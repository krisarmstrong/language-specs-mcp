# HTMLMeterElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨November 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FHTMLMeterElement&level=high)

The HTML [<meter>](/en-US/docs/Web/HTML/Reference/Elements/meter) elements expose the `HTMLMeterElement` interface, which provides special properties and methods (beyond the [HTMLElement](/en-US/docs/Web/API/HTMLElement) object interface they also have available to them by inheritance) for manipulating the layout and presentation of [<meter>](/en-US/docs/Web/HTML/Reference/Elements/meter) elements.

## [Instance properties](#instance_properties)

Also inherits properties from its parent, [HTMLElement](/en-US/docs/Web/API/HTMLElement).

[HTMLMeterElement.high](/en-US/docs/Web/API/HTMLMeterElement/high)

A `double` representing the value of the high boundary, reflecting the [high](/en-US/docs/Web/HTML/Reference/Elements/meter#high) attribute.

[HTMLMeterElement.low](/en-US/docs/Web/API/HTMLMeterElement/low)

A `double` representing the value of the low boundary, reflecting the [low](/en-US/docs/Web/HTML/Reference/Elements/meter#low) attribute.

[HTMLMeterElement.max](/en-US/docs/Web/API/HTMLMeterElement/max)

A `double` representing the maximum value, reflecting the [max](/en-US/docs/Web/HTML/Reference/Elements/meter#max) attribute.

[HTMLMeterElement.min](/en-US/docs/Web/API/HTMLMeterElement/min)

A `double` representing the minimum value, reflecting the [min](/en-US/docs/Web/HTML/Reference/Elements/meter#min) attribute.

[HTMLMeterElement.optimum](/en-US/docs/Web/API/HTMLMeterElement/optimum)

A `double` representing the optimum, reflecting the [optimum](/en-US/docs/Web/HTML/Reference/Elements/meter#optimum) attribute.

[HTMLMeterElement.value](/en-US/docs/Web/API/HTMLMeterElement/value)

A `double` representing the current value, reflecting the [value](/en-US/docs/Web/HTML/Reference/Elements/meter#value) attribute.

[HTMLMeterElement.labels](/en-US/docs/Web/API/HTMLMeterElement/labels)Read only

A [NodeList](/en-US/docs/Web/API/NodeList) of [<label>](/en-US/docs/Web/HTML/Reference/Elements/label) elements that are associated with the element.

## [Instance methods](#instance_methods)

This interface does not implement any specific methods but inherits methods from its parent, [HTMLElement](/en-US/docs/Web/API/HTMLElement).
