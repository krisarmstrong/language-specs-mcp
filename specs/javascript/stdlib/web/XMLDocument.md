# XMLDocument

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨September 2016⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FXMLDocument&level=high)

The XMLDocument interface represents an XML document. It inherits from the generic [Document](/en-US/docs/Web/API/Document) and does not add any specific methods or properties to it: nevertheless, several algorithms behave differently with the two types of documents.
