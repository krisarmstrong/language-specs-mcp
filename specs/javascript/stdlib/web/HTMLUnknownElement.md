# HTMLUnknownElement

The `HTMLUnknownElement` interface represents an invalid HTML element and derives from the [HTMLElement](/en-US/docs/Web/API/HTMLElement) interface, but without implementing any additional properties or methods.

## [Instance properties](#instance_properties)
