# GPUCompilationMessage

Limited availability

This feature is not Baseline because it does not work in some of the most widely-used browsers.

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FGPUCompilationMessage&level=not)

Secure context: This feature is available only in [secure contexts](/en-US/docs/Web/Security/Defenses/Secure_Contexts) (HTTPS), in some or all [supporting browsers](#browser_compatibility).

Note: This feature is available in [Web Workers](/en-US/docs/Web/API/Web_Workers_API).

The `GPUCompilationMessage` interface of the [WebGPU API](/en-US/docs/Web/API/WebGPU_API) represents a single informational, warning, or error message generated by the GPU shader module compiler.

An array of `GPUCompilationMessage` objects is available in the `messages` property of the [GPUCompilationInfo](/en-US/docs/Web/API/GPUCompilationInfo) object accessed via [GPUShaderModule.getCompilationInfo()](/en-US/docs/Web/API/GPUShaderModule/getCompilationInfo).

## [Instance properties](#instance_properties)

[length](/en-US/docs/Web/API/GPUCompilationMessage/length)Read only

A number representing the length of the substring that the message corresponds to.

[lineNum](/en-US/docs/Web/API/GPUCompilationMessage/lineNum)Read only

A number representing the line number in the shader code that the message corresponds to.

[linePos](/en-US/docs/Web/API/GPUCompilationMessage/linePos)Read only

A number representing the position in the code line that the message corresponds to. This could be an exact point, or the start of the relevant substring.

[message](/en-US/docs/Web/API/GPUCompilationMessage/message)Read only

A string representing human-readable message text.

[offset](/en-US/docs/Web/API/GPUCompilationMessage/offset)Read only

A number representing the offset from the start of the shader code to the exact point, or the start of the relevant substring, that the message corresponds to.

[type](/en-US/docs/Web/API/GPUCompilationMessage/type)Read only

An enumerated value representing the type of the message — `"error"`, `"info"`, or `"warning"`.
