# AnimationTimeline

 Baseline  Widely available  * 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2020⁩. 

* Some parts of this feature may have varying levels of support.

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FAnimationTimeline&level=high)

The `AnimationTimeline` interface of the [Web Animations API](/en-US/docs/Web/API/Web_Animations_API) represents the timeline of an animation. This interface exists to define timeline features, inherited by other timeline types:

- [DocumentTimeline](/en-US/docs/Web/API/DocumentTimeline)
- [ScrollTimeline](/en-US/docs/Web/API/ScrollTimeline)
- [ViewTimeline](/en-US/docs/Web/API/ViewTimeline)

## [Instance properties](#instance_properties)

[currentTime](/en-US/docs/Web/API/AnimationTimeline/currentTime)Read only

Returns the time value in milliseconds for this timeline or `null` if this timeline is inactive.

[duration](/en-US/docs/Web/API/AnimationTimeline/duration)Read only

Returns the maximum value for this timeline or `null`.
