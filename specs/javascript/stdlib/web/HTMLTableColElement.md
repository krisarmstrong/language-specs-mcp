# HTMLTableColElement

 Baseline  Widely available  * 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

* Some parts of this feature may have varying levels of support.

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FHTMLTableColElement&level=high)

The `HTMLTableColElement` interface provides properties for manipulating single or grouped table column elements.

## [Instance properties](#instance_properties)

Inherits properties from its parent, [HTMLElement](/en-US/docs/Web/API/HTMLElement).

[HTMLTableColElement.align](/en-US/docs/Web/API/HTMLTableColElement/align)Deprecated

A string that indicates the horizontal alignment of the cell data in the column.

[HTMLTableColElement.ch](/en-US/docs/Web/API/HTMLTableColElement/ch)Deprecated

A string representing the alignment character for cell data.

[HTMLTableColElement.chOff](/en-US/docs/Web/API/HTMLTableColElement/chOff)Deprecated

A string representing the offset for the alignment character.

[HTMLTableColElement.span](/en-US/docs/Web/API/HTMLTableColElement/span)

A positive number that reflects the [span](/en-US/docs/Web/HTML/Reference/Elements/col#span) HTML attribute, indicating the number of columns to apply this object's attributes to.

[HTMLTableColElement.vAlign](/en-US/docs/Web/API/HTMLTableColElement/vAlign)Deprecated

A string that indicates the vertical alignment of the cell data in the column.

`HTMLTableColElement.width`Deprecated

A string representing the default column width.

## [Instance methods](#instance_methods)

No specific method; inherits methods from its parent, [HTMLElement](/en-US/docs/Web/API/HTMLElement).
