# HTMLDataElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨October 2017⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FHTMLDataElement&level=high)

The `HTMLDataElement` interface provides special properties (beyond the regular [HTMLElement](/en-US/docs/Web/API/HTMLElement) interface it also has available to it by inheritance) for manipulating [<data>](/en-US/docs/Web/HTML/Reference/Elements/data) elements.

## [Instance properties](#instance_properties)

Inherits properties from its parent, [HTMLElement](/en-US/docs/Web/API/HTMLElement).

[HTMLDataElement.value](/en-US/docs/Web/API/HTMLDataElement/value)

A string reflecting the [value](/en-US/docs/Web/HTML/Reference/Elements/data#value) HTML attribute, containing a machine-readable form of the element's value.

## [Instance methods](#instance_methods)

No specific method; inherits methods from its parent, [HTMLElement](/en-US/docs/Web/API/HTMLElement).
