# SVGFEDiffuseLightingElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGFEDiffuseLightingElement&level=high)

The `SVGFEDiffuseLightingElement` interface corresponds to the [<feDiffuseLighting>](/en-US/docs/Web/SVG/Reference/Element/feDiffuseLighting) element.

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent interface, [SVGElement](/en-US/docs/Web/API/SVGElement).

[SVGFEDiffuseLightingElement.diffuseConstant](/en-US/docs/Web/API/SVGFEDiffuseLightingElement/diffuseConstant)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [diffuseConstant](/en-US/docs/Web/SVG/Reference/Attribute/diffuseConstant) attribute of the given element.

[SVGFEDiffuseLightingElement.height](/en-US/docs/Web/API/SVGFEDiffuseLightingElement/height)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [height](/en-US/docs/Web/SVG/Reference/Attribute/height) attribute of the given element.

[SVGFEDiffuseLightingElement.in1](/en-US/docs/Web/API/SVGFEDiffuseLightingElement/in1)Read only

An [SVGAnimatedString](/en-US/docs/Web/API/SVGAnimatedString) corresponding to the [in](/en-US/docs/Web/SVG/Reference/Attribute/in) attribute of the given element.

[SVGFEDiffuseLightingElement.kernelUnitLengthX](/en-US/docs/Web/API/SVGFEDiffuseLightingElement/kernelUnitLengthX)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the X component of the [kernelUnitLength](/en-US/docs/Web/SVG/Reference/Attribute/kernelUnitLength) attribute of the given element.

[SVGFEDiffuseLightingElement.kernelUnitLengthY](/en-US/docs/Web/API/SVGFEDiffuseLightingElement/kernelUnitLengthY)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the Y component of the [kernelUnitLength](/en-US/docs/Web/SVG/Reference/Attribute/kernelUnitLength) attribute of the given element.

[SVGFEDiffuseLightingElement.result](/en-US/docs/Web/API/SVGFEDiffuseLightingElement/result)Read only

An [SVGAnimatedString](/en-US/docs/Web/API/SVGAnimatedString) corresponding to the [result](/en-US/docs/Web/SVG/Reference/Attribute/result) attribute of the given element.

[SVGFEDiffuseLightingElement.surfaceScale](/en-US/docs/Web/API/SVGFEDiffuseLightingElement/surfaceScale)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [surfaceScale](/en-US/docs/Web/SVG/Reference/Attribute/surfaceScale) attribute of the given element.

[SVGFEDiffuseLightingElement.width](/en-US/docs/Web/API/SVGFEDiffuseLightingElement/width)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [width](/en-US/docs/Web/SVG/Reference/Attribute/width) attribute of the given element.

[SVGFEDiffuseLightingElement.x](/en-US/docs/Web/API/SVGFEDiffuseLightingElement/x)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [x](/en-US/docs/Web/SVG/Reference/Attribute/x) attribute of the given element.

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [y](/en-US/docs/Web/SVG/Reference/Attribute/y) attribute of the given element.
