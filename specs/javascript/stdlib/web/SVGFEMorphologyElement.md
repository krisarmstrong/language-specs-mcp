# SVGFEMorphologyElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGFEMorphologyElement&level=high)

The `SVGFEMorphologyElement` interface corresponds to the [<feMorphology>](/en-US/docs/Web/SVG/Reference/Element/feMorphology) element.

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent interface, [SVGElement](/en-US/docs/Web/API/SVGElement).

[SVGFEMorphologyElement.height](/en-US/docs/Web/API/SVGFEMorphologyElement/height)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [height](/en-US/docs/Web/SVG/Reference/Attribute/height) attribute of the given element.

[SVGFEMorphologyElement.in1](/en-US/docs/Web/API/SVGFEMorphologyElement/in1)Read only

An [SVGAnimatedString](/en-US/docs/Web/API/SVGAnimatedString) corresponding to the [in](/en-US/docs/Web/SVG/Reference/Attribute/in) attribute of the given element.

[SVGFEMorphologyElement.operator](/en-US/docs/Web/API/SVGFEMorphologyElement/operator)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [operator](/en-US/docs/Web/SVG/Reference/Attribute/operator) attribute of the given element. It takes one of the `SVG_MORPHOLOGY_OPERATOR_*` constants defined on this interface.

[SVGFEMorphologyElement.radiusX](/en-US/docs/Web/API/SVGFEMorphologyElement/radiusX)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the X component of the [radius](/en-US/docs/Web/SVG/Reference/Attribute/radius) attribute of the given element.

[SVGFEMorphologyElement.radiusY](/en-US/docs/Web/API/SVGFEMorphologyElement/radiusY)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the Y component of the [radius](/en-US/docs/Web/SVG/Reference/Attribute/radius) attribute of the given element.

[SVGFEMorphologyElement.result](/en-US/docs/Web/API/SVGFEMorphologyElement/result)Read only

An [SVGAnimatedString](/en-US/docs/Web/API/SVGAnimatedString) corresponding to the [result](/en-US/docs/Web/SVG/Reference/Attribute/result) attribute of the given element.

[SVGFEMorphologyElement.width](/en-US/docs/Web/API/SVGFEMorphologyElement/width)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [width](/en-US/docs/Web/SVG/Reference/Attribute/width) attribute of the given element.

[SVGFEMorphologyElement.x](/en-US/docs/Web/API/SVGFEMorphologyElement/x)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [x](/en-US/docs/Web/SVG/Reference/Attribute/x) attribute of the given element.

[SVGFEMorphologyElement.y](/en-US/docs/Web/API/SVGFEMorphologyElement/y)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [y](/en-US/docs/Web/SVG/Reference/Attribute/y) attribute of the given element.

## [Instance methods](#instance_methods)

This interface does not provide any specific methods, but implements those of its parent, [SVGElement](/en-US/docs/Web/API/SVGElement).

## [Static properties](#static_properties)

[SVG_MORPHOLOGY_OPERATOR_UNKNOWN (0)](#svg_morphology_operator_unknown)

The type is not one of predefined types. It is invalid to attempt to define a new value of this type or to attempt to switch an existing value to this type.

[SVG_MORPHOLOGY_OPERATOR_ERODE (1)](#svg_morphology_operator_erode)

Corresponds to the value `erode`.

[SVG_MORPHOLOGY_OPERATOR_DILATE (2)](#svg_morphology_operator_dilate)

Corresponds to the value `dilate`.
