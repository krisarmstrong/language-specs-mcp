# SVGFESpotLightElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGFESpotLightElement&level=high)

The `SVGFESpotLightElement` interface corresponds to the [<feSpotLight>](/en-US/docs/Web/SVG/Reference/Element/feSpotLight) element.

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent interface, [SVGElement](/en-US/docs/Web/API/SVGElement).

[SVGFESpotLightElement.x](/en-US/docs/Web/API/SVGFESpotLightElement/x)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [x](/en-US/docs/Web/SVG/Reference/Attribute/x) attribute of the given element.

[SVGFESpotLightElement.y](/en-US/docs/Web/API/SVGFESpotLightElement/y)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [y](/en-US/docs/Web/SVG/Reference/Attribute/y) attribute of the given element.

[SVGFESpotLightElement.z](/en-US/docs/Web/API/SVGFESpotLightElement/z)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [z](/en-US/docs/Web/SVG/Reference/Attribute/z) attribute of the given element.

[SVGFESpotLightElement.pointsAtX](/en-US/docs/Web/API/SVGFESpotLightElement/pointsAtX)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [pointsAtX](/en-US/docs/Web/SVG/Reference/Attribute/pointsAtX) attribute of the given element.

[SVGFESpotLightElement.pointsAtY](/en-US/docs/Web/API/SVGFESpotLightElement/pointsAtY)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [pointsAtY](/en-US/docs/Web/SVG/Reference/Attribute/pointsAtY) attribute of the given element.

[SVGFESpotLightElement.pointsAtZ](/en-US/docs/Web/API/SVGFESpotLightElement/pointsAtZ)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [pointsAtZ](/en-US/docs/Web/SVG/Reference/Attribute/pointsAtZ) attribute of the given element.

[SVGFESpotLightElement.specularExponent](/en-US/docs/Web/API/SVGFESpotLightElement/specularExponent)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [specularExponent](/en-US/docs/Web/SVG/Reference/Attribute/specularExponent) attribute of the given element.

[SVGFESpotLightElement.limitingConeAngle](/en-US/docs/Web/API/SVGFESpotLightElement/limitingConeAngle)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [limitingConeAngle](/en-US/docs/Web/SVG/Reference/Attribute/limitingConeAngle) attribute of the given element.

## [Instance methods](#instance_methods)

This interface does not provide any specific methods, but implements those of its parent, [SVGElement](/en-US/docs/Web/API/SVGElement).
