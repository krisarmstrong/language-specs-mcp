# SVGTextPositioningElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGTextPositioningElement&level=high)

The `SVGTextPositioningElement` interface is implemented by elements that support attributes that position individual text glyphs. It is inherited by [SVGTextElement](/en-US/docs/Web/API/SVGTextElement) and [SVGTSpanElement](/en-US/docs/Web/API/SVGTSpanElement).

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent, [SVGTextContentElement](/en-US/docs/Web/API/SVGTextContentElement).

[SVGTextPositioningElement.x](/en-US/docs/Web/API/SVGTextPositioningElement/x)Read only

Returns an [SVGAnimatedLengthList](/en-US/docs/Web/API/SVGAnimatedLengthList) reflecting the [x](/en-US/docs/Web/SVG/Reference/Attribute/x) attribute of the given element.

[SVGTextPositioningElement.y](/en-US/docs/Web/API/SVGTextPositioningElement/y)Read only

Returns an [SVGAnimatedLengthList](/en-US/docs/Web/API/SVGAnimatedLengthList) reflecting the [y](/en-US/docs/Web/SVG/Reference/Attribute/y) attribute of the given element.

[SVGTextPositioningElement.dx](/en-US/docs/Web/API/SVGTextPositioningElement/dx)Read only

Returns an [SVGAnimatedLengthList](/en-US/docs/Web/API/SVGAnimatedLengthList) reflecting the [dx](/en-US/docs/Web/SVG/Reference/Attribute/dx) attribute of the given element.

[SVGTextPositioningElement.dy](/en-US/docs/Web/API/SVGTextPositioningElement/dy)Read only

Returns an [SVGAnimatedLengthList](/en-US/docs/Web/API/SVGAnimatedLengthList) reflecting the [dy](/en-US/docs/Web/SVG/Reference/Attribute/dy) attribute of the given element.

[SVGTextPositioningElement.rotate](/en-US/docs/Web/API/SVGTextPositioningElement/rotate)Read only

Returns an [SVGAnimatedNumberList](/en-US/docs/Web/API/SVGAnimatedNumberList) reflecting the [rotate](/en-US/docs/Web/SVG/Reference/Attribute/rotate) attribute of the given element.

## [Instance methods](#instance_methods)

This interface doesn't provide any specific methods, but inherits methods from its parent, [SVGTextContentElement](/en-US/docs/Web/API/SVGTextContentElement).
