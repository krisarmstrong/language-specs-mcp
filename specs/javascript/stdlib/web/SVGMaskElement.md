# SVGMaskElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGMaskElement&level=high)

The `SVGMaskElement` interface provides access to the properties of [<mask>](/en-US/docs/Web/SVG/Reference/Element/mask) elements, as well as methods to manipulate them.

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent, [SVGElement](/en-US/docs/Web/API/SVGElement).

[SVGMaskElement.maskUnits](/en-US/docs/Web/API/SVGMaskElement/maskUnits)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [maskUnits](/en-US/docs/Web/SVG/Reference/Attribute/maskUnits) attribute of the given [<mask>](/en-US/docs/Web/SVG/Reference/Element/mask) element. Takes one of the constants defined in [SVGUnitTypes](/en-US/docs/Web/API/SVGUnitTypes).

[SVGMaskElement.maskContentUnits](/en-US/docs/Web/API/SVGMaskElement/maskContentUnits)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [maskContentUnits](/en-US/docs/Web/SVG/Reference/Attribute/maskContentUnits) attribute of the given [<mask>](/en-US/docs/Web/SVG/Reference/Element/mask) element. Takes one of the constants defined in [SVGUnitTypes](/en-US/docs/Web/API/SVGUnitTypes).

[SVGMaskElement.x](/en-US/docs/Web/API/SVGMaskElement/x)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [x](/en-US/docs/Web/SVG/Reference/Attribute/x) attribute of the given [<mask>](/en-US/docs/Web/SVG/Reference/Element/mask) element.

[SVGMaskElement.y](/en-US/docs/Web/API/SVGMaskElement/y)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [y](/en-US/docs/Web/SVG/Reference/Attribute/y) attribute of the given [<mask>](/en-US/docs/Web/SVG/Reference/Element/mask) element.

[SVGMaskElement.width](/en-US/docs/Web/API/SVGMaskElement/width)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [width](/en-US/docs/Web/SVG/Reference/Attribute/width) attribute of the given [<mask>](/en-US/docs/Web/SVG/Reference/Element/mask) element.

[SVGMaskElement.height](/en-US/docs/Web/API/SVGMaskElement/height)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [height](/en-US/docs/Web/SVG/Reference/Attribute/height) attribute of the given [<mask>](/en-US/docs/Web/SVG/Reference/Element/mask) element.

## [Instance methods](#instance_methods)

This interface doesn't implement any specific methods, but inherits methods from its parent interface, [SVGElement](/en-US/docs/Web/API/SVGElement).
