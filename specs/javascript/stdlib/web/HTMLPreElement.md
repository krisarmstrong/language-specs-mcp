# HTMLPreElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FHTMLPreElement&level=high)

The `HTMLPreElement` interface exposes specific properties and methods (beyond those of the [HTMLElement](/en-US/docs/Web/API/HTMLElement) interface it also has available to it by inheritance) for manipulating a block of preformatted text ([<pre>](/en-US/docs/Web/HTML/Reference/Elements/pre)).

## [Instance properties](#instance_properties)

Inherits properties from its parent, [HTMLElement](/en-US/docs/Web/API/HTMLElement).

`HTMLPreElement.width`Deprecated

A `long` value reflecting the obsolete [width](/en-US/docs/Web/HTML/Reference/Elements/pre#width) attribute, containing a fixed-size length for the [<pre>](/en-US/docs/Web/HTML/Reference/Elements/pre) element.

## [Instance methods](#instance_methods)

No specific method; inherits methods from its parent, [HTMLElement](/en-US/docs/Web/API/HTMLElement).
