# SVGLineElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGLineElement&level=high)

The `SVGLineElement` interface provides access to the properties of [<line>](/en-US/docs/Web/SVG/Reference/Element/line) elements, as well as methods to manipulate them.

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent, [SVGGeometryElement](/en-US/docs/Web/API/SVGGeometryElement).

[SVGLineElement.x1](/en-US/docs/Web/API/SVGLineElement/x1)Read only

Returns an [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) that corresponds to attribute [x1](/en-US/docs/Web/SVG/Reference/Attribute/x1) on the given [<line>](/en-US/docs/Web/SVG/Reference/Element/line) element.

[SVGLineElement.y1](/en-US/docs/Web/API/SVGLineElement/y1)Read only

Returns an [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) that corresponds to attribute [y1](/en-US/docs/Web/SVG/Reference/Attribute/y1) on the given [<line>](/en-US/docs/Web/SVG/Reference/Element/line) element.

[SVGLineElement.x2](/en-US/docs/Web/API/SVGLineElement/x2)Read only

Returns an [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) that corresponds to attribute [x2](/en-US/docs/Web/SVG/Reference/Attribute/x2) on the given [<line>](/en-US/docs/Web/SVG/Reference/Element/line) element.

[SVGLineElement.y2](/en-US/docs/Web/API/SVGLineElement/y2)Read only

Returns an [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) that corresponds to attribute [y2](/en-US/docs/Web/SVG/Reference/Attribute/y2) on the given [<line>](/en-US/docs/Web/SVG/Reference/Element/line) element.
