# SVGFETurbulenceElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGFETurbulenceElement&level=high)

The `SVGFETurbulenceElement` interface corresponds to the [<feTurbulence>](/en-US/docs/Web/SVG/Reference/Element/feTurbulence) element.

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent interface, [SVGElement](/en-US/docs/Web/API/SVGElement).

[SVGFETurbulenceElement.baseFrequencyX](/en-US/docs/Web/API/SVGFETurbulenceElement/baseFrequencyX)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the X component of the [baseFrequency](/en-US/docs/Web/SVG/Reference/Attribute/baseFrequency) attribute of the given element.

[SVGFETurbulenceElement.baseFrequencyY](/en-US/docs/Web/API/SVGFETurbulenceElement/baseFrequencyY)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the Y component of the [baseFrequency](/en-US/docs/Web/SVG/Reference/Attribute/baseFrequency) attribute of the given element.

[SVGFETurbulenceElement.height](/en-US/docs/Web/API/SVGFETurbulenceElement/height)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [height](/en-US/docs/Web/SVG/Reference/Attribute/height) attribute of the given element.

[SVGFETurbulenceElement.numOctaves](/en-US/docs/Web/API/SVGFETurbulenceElement/numOctaves)Read only

An [SVGAnimatedInteger](/en-US/docs/Web/API/SVGAnimatedInteger) corresponding to the [numOctaves](/en-US/docs/Web/SVG/Reference/Attribute/numOctaves) attribute of the given element.

[SVGFETurbulenceElement.result](/en-US/docs/Web/API/SVGFETurbulenceElement/result)Read only

An [SVGAnimatedString](/en-US/docs/Web/API/SVGAnimatedString) corresponding to the [result](/en-US/docs/Web/SVG/Reference/Attribute/result) attribute of the given element.

[SVGFETurbulenceElement.seed](/en-US/docs/Web/API/SVGFETurbulenceElement/seed)Read only

An [SVGAnimatedNumber](/en-US/docs/Web/API/SVGAnimatedNumber) corresponding to the [seed](/en-US/docs/Web/SVG/Reference/Attribute/seed) attribute of the given element.

[SVGFETurbulenceElement.stitchTiles](/en-US/docs/Web/API/SVGFETurbulenceElement/stitchTiles)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [stitchTiles](/en-US/docs/Web/SVG/Reference/Attribute/stitchTiles) attribute of the given element. It takes one of the `SVG_STITCHTYPE_*` constants defined on this interface.

[SVGFETurbulenceElement.type](/en-US/docs/Web/API/SVGFETurbulenceElement/type)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [type](/en-US/docs/Web/SVG/Reference/Attribute/type) attribute of the given element. It takes one of the `SVG_TURBULENCE_TYPE_*` constants defined on this interface.

[SVGFETurbulenceElement.width](/en-US/docs/Web/API/SVGFETurbulenceElement/width)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [width](/en-US/docs/Web/SVG/Reference/Attribute/width) attribute of the given element.

[SVGFETurbulenceElement.x](/en-US/docs/Web/API/SVGFETurbulenceElement/x)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [x](/en-US/docs/Web/SVG/Reference/Attribute/x) attribute of the given element.

[SVGFETurbulenceElement.y](/en-US/docs/Web/API/SVGFETurbulenceElement/y)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [y](/en-US/docs/Web/SVG/Reference/Attribute/y) attribute of the given element.

## [Instance methods](#instance_methods)

This interface does not provide any specific methods, but implements those of its parent, [SVGElement](/en-US/docs/Web/API/SVGElement).

## [Static properties](#static_properties)

[SVG_TURBULENCE_TYPE_UNKNOWN (0)](#svg_turbulence_type_unknown)

The type is not one of predefined types. It is invalid to attempt to define a new value of this type or to attempt to switch an existing value to this type.

[SVG_TURBULENCE_TYPE_FRACTALNOISE (1)](#svg_turbulence_type_fractalnoise)

Corresponds to the value `fractalNoise`.

[SVG_TURBULENCE_TYPE_TURBULENCE (2)](#svg_turbulence_type_turbulence)

Corresponds to the value `turbulence`.

[SVG_STITCHTYPE_UNKNOWN (0)](#svg_stitchtype_unknown)

The type is not one of predefined types. It is invalid to attempt to define a new value of this type or to attempt to switch an existing value to this type.

[SVG_STITCHTYPE_STITCH (1)](#svg_stitchtype_stitch)

Corresponds to the value `stitch`.

[SVG_STITCHTYPE_NOSTITCH (2)](#svg_stitchtype_nostitch)

Corresponds to the value `noStitch`.
