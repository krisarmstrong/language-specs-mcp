# SVGFEFuncAElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGFEFuncAElement&level=high)

The `SVGFEFuncAElement` interface corresponds to the [<feFuncA>](/en-US/docs/Web/SVG/Reference/Element/feFuncA) element.
