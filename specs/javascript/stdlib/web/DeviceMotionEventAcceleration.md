# DeviceMotionEventAcceleration

Limited availability

This feature is not Baseline because it does not work in some of the most widely-used browsers.

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FDeviceMotionEventAcceleration&level=not)

Secure context: This feature is available only in [secure contexts](/en-US/docs/Web/Security/Defenses/Secure_Contexts) (HTTPS), in some or all [supporting browsers](#browser_compatibility).

The `DeviceMotionEventAcceleration` interface of the [Device Orientation Events](/en-US/docs/Web/API/Device_orientation_events) provides information about the amount of acceleration the device is experiencing along all three axes.

## [Instance properties](#instance_properties)

[DeviceMotionEventAcceleration.x](/en-US/docs/Web/API/DeviceMotionEventAcceleration/x)Read only

The amount of acceleration along the X axis.

[DeviceMotionEventAcceleration.y](/en-US/docs/Web/API/DeviceMotionEventAcceleration/y)Read only

The amount of acceleration along the Y axis.

[DeviceMotionEventAcceleration.z](/en-US/docs/Web/API/DeviceMotionEventAcceleration/z)Read only

The amount of acceleration along the Z axis.
