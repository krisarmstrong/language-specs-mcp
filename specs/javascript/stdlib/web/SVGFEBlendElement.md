# SVGFEBlendElement

 Baseline  Widely available 

 This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨July 2015⁩. 

- [Learn more](/en-US/docs/Glossary/Baseline/Compatibility)
- [See full compatibility](#browser_compatibility)
- [Report feedback](https://survey.alchemer.com/s3/7634825/MDN-baseline-feedback?page=%2Fen-US%2Fdocs%2FWeb%2FAPI%2FSVGFEBlendElement&level=high)

The `SVGFEBlendElement` interface corresponds to the [<feBlend>](/en-US/docs/Web/SVG/Reference/Element/feBlend) element.

## [Instance properties](#instance_properties)

This interface also inherits properties from its parent interface, [SVGElement](/en-US/docs/Web/API/SVGElement).

[SVGFEBlendElement.height](/en-US/docs/Web/API/SVGFEBlendElement/height)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [height](/en-US/docs/Web/SVG/Reference/Attribute/height) attribute of the given element.

[SVGFEBlendElement.in1](/en-US/docs/Web/API/SVGFEBlendElement/in1)Read only

An [SVGAnimatedString](/en-US/docs/Web/API/SVGAnimatedString) corresponding to the [in](/en-US/docs/Web/SVG/Reference/Attribute/in) attribute of the given element.

[SVGFEBlendElement.in2](/en-US/docs/Web/API/SVGFEBlendElement/in2)Read only

An [SVGAnimatedString](/en-US/docs/Web/API/SVGAnimatedString) corresponding to the [in2](/en-US/docs/Web/SVG/Reference/Attribute/in2) attribute of the given element.

[SVGFEBlendElement.mode](/en-US/docs/Web/API/SVGFEBlendElement/mode)Read only

An [SVGAnimatedEnumeration](/en-US/docs/Web/API/SVGAnimatedEnumeration) corresponding to the [mode](/en-US/docs/Web/SVG/Reference/Attribute/mode) attribute of the given element. It takes one of the `SVG_FEBLEND_MODE_*` constants defined on this interface.

[SVGFEBlendElement.result](/en-US/docs/Web/API/SVGFEBlendElement/result)Read only

An [SVGAnimatedString](/en-US/docs/Web/API/SVGAnimatedString) corresponding to the [result](/en-US/docs/Web/SVG/Reference/Attribute/result) attribute of the given element.

[SVGFEBlendElement.width](/en-US/docs/Web/API/SVGFEBlendElement/width)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [width](/en-US/docs/Web/SVG/Reference/Attribute/width) attribute of the given element.

[SVGFEBlendElement.x](/en-US/docs/Web/API/SVGFEBlendElement/x)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [x](/en-US/docs/Web/SVG/Reference/Attribute/x) attribute of the given element.

[SVGFEBlendElement.y](/en-US/docs/Web/API/SVGFEBlendElement/y)Read only

An [SVGAnimatedLength](/en-US/docs/Web/API/SVGAnimatedLength) corresponding to the [y](/en-US/docs/Web/SVG/Reference/Attribute/y) attribute of the given element.

## [Instance methods](#instance_methods)

This interface does not provide any specific methods, but implements those of its parent, [SVGElement](/en-US/docs/Web/API/SVGElement).

## [Static properties](#static_properties)

[SVG_FEBLEND_MODE_UNKNOWN (0)](#svg_feblend_mode_unknown)

The type is not one of predefined types. It is invalid to attempt to define a new value of this type or to attempt to switch an existing value to this type.

[SVG_FEBLEND_MODE_NORMAL (1)](#svg_feblend_mode_normal)

Corresponds to the value `normal`.

[SVG_FEBLEND_MODE_MULTIPLY (2)](#svg_feblend_mode_multiply)

Corresponds to the value `multiply`.

[SVG_FEBLEND_MODE_SCREEN (3)](#svg_feblend_mode_screen)

Corresponds to the value `screen`.

[SVG_FEBLEND_MODE_DARKEN (4)](#svg_feblend_mode_darken)

Corresponds to the value `darken`.

[SVG_FEBLEND_MODE_LIGHTEN (5)](#svg_feblend_mode_lighten)

Corresponds to the value `lighten`.
