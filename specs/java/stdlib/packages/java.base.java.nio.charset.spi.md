# Package java.nio.charset.spi

package java.nio.charset.spiService-provider classes for the [java.nio.charset](../package-summary.html) package. 

 Only developers who are defining new charsets should need to make direct use of this package. 

 Unless otherwise noted, passing a `null` argument to a constructor or method in any class or interface in this package will cause a [NullPointerException](../../../lang/NullPointerException.html) to be thrown.
