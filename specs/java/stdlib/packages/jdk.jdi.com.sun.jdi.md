# Package com.sun.jdi

package com.sun.jdiThis is the core package of the Java Debug Interface (JDI), it defines mirrors for values, types, and the target VirtualMachine itself - as well bootstrapping facilities. [VirtualMachine](VirtualMachine.html) mirrors the target virtual machine and is the origin of all information provided by the JDI. A VirtualMachine is typically created by using the [VirtualMachineManager](VirtualMachineManager.html) to create a connection to the target virtual machine (see the [com.sun.jdi.connect](connect/package-summary.html) package). In turn the [VirtualMachineManager](VirtualMachineManager.html) is typically created by calling [Bootstrap.virtualMachineManager()](Bootstrap.html#virtualMachineManager()). 

 Most of the methods within this package can throw the unchecked exception [VMDisconnectedException](VMDisconnectedException.html). 

 Methods may be added to the interfaces in the JDI packages in future releases. Existing packages may be renamed if the JDI becomes a standard extension.
