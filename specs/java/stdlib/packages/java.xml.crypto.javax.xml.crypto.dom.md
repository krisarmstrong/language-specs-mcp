# Package javax.xml.crypto.dom

package javax.xml.crypto.domDOM-specific classes for the [javax.xml.crypto](../package-summary.html) package. Only users who are using DOM-based XML cryptographic implementations (ex: [XMLSignatureFactory](../dsig/XMLSignatureFactory.html) or [KeyInfoFactory](../dsig/keyinfo/KeyInfoFactory.html)) should need to make direct use of this package.
