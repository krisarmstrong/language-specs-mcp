# Package org.w3c.dom.css

package org.w3c.dom.cssProvides interfaces for DOM Level 2 Style Specification. Refer to the [Document Object Model (DOM) Level 2 Style Specification](http://www.w3.org/TR/2000/REC-DOM-Level-2-Style-20001113), the CSS and stylesheets modules define the Document Object Model Level 2 Style Sheets and Cascading Style Sheets (CSS) that allow programs and scripts to dynamically access and update the content of style sheets documents.Since:1.4
