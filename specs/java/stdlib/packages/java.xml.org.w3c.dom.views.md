# Package org.w3c.dom.views

package org.w3c.dom.viewsProvides interfaces for DOM Level 2 Views. Refer to the [Document Object Model (DOM) Level 2 Views Specification](http://www.w3.org/TR/2000/REC-DOM-Level-2-Views-20001113), the Views module allows programs and scripts to dynamically access and update the content of a representation of a document.Since:1.8
