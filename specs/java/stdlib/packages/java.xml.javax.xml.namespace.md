# Package javax.xml.namespace

package javax.xml.namespace

 Defines XML Namespace processing. 

 The following XML standards apply: 

- [XML Schema Part2: Datatypes specification](http://www.w3.org/TR/xmlschema-2/#QName)
- [Namespaces in XML](https://www.w3.org/TR/REC-xml-names/)
- [Namespaces in XML Errata](https://www.w3.org/XML/xml-names-19990114-errata.html)
