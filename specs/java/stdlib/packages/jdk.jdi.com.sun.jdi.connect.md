# Package com.sun.jdi.connect

package com.sun.jdi.connectThis package defines connections between the virtual machine using the JDI and the target virtual machine. In concert with [VirtualMachineManager](../VirtualMachineManager.html) it is the mechanism for launching, attaching, etc to target virtual machines. 

 Methods may be added to the interfaces in the JDI packages in future releases. Existing packages may be renamed if the JDI becomes a standard extension.
