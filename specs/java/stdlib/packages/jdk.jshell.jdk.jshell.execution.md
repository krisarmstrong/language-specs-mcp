# Package jdk.jshell.execution

package jdk.jshell.executionProvides implementation support for building JShell execution engines. Includes concrete and abstract implementations of the [ExecutionControl](../spi/ExecutionControl.html) interface. Also, provides related communication utilities. This package may be used to define alternative execution engines. The default JShell execution engine is included.Since:9
