# Package javax.xml.transform.stax

package javax.xml.transform.staxProvides StAX specific transformation classes. 

 The [StAXSource](StAXSource.html) class encapsulates a [XMLStreamReader](../../stream/XMLStreamReader.html) or [XMLEventReader](../../stream/XMLEventReader.html) and can be used as an input where a [Source](../Source.html) object is accepted. 

 The [StAXResult](StAXResult.html) class encapsulates a [XMLStreamWriter](../../stream/XMLStreamWriter.html) or [XMLEventWriter](../../stream/XMLEventWriter.html) and can be used as an output where a [Result](../Result.html) object is accepted.
