# Package org.w3c.dom.html

package org.w3c.dom.htmlProvides interfaces for DOM Level 2 HTML Specification. Refer to the [Document Object Model (DOM) Level 2 Specification](http://www.w3.org/TR/2000/CR-DOM-Level-2-20000510), the HTML module extends the Core API to describe objects and methods specific to HTML documents that comply with [HTML 4.0 Specification](https://www.w3.org/TR/1998/REC-html40-19980424/).Since:1.4
