Search across all language specifications for a term or concept.

Arguments:
- $ARGUMENTS: Search query (can be multiple words)

Use search_specs with the provided query to find relevant documentation across all supported languages.

Examples:
- /search-specs error handling
- /search-specs async await
- /search-specs null safety
- /search-specs dependency injection

Full query: $ARGUMENTS
